1. What Brigades Do

What is a Brigade

A Brigade is a local chapter of the Code for America Brigade network. The Code for America Brigade network is an international citizen volunteer program that creates new ways for residents to participate in building stronger local communities, in collaboration with government.

A group can join the Code for America Brigade network as a Brigade Network Member or an Official Brigade Chapter.

Brigade Network Member

  • A Brigade Network Member is a local Brigade chapter committed to organizing people and participating in the Code for America Brigade network.
  • Brigade Network Members have a leadership teams and hold consistent meetings (or “hack nights”).
  • Brigade Network members have access to Code for America trainings and communication channels, allowing them learn and ask question alongside other leaders throughout the network.
  • If interested, a Brigade Network Member can become an Official Brigade Chapter by working with municipal and community partners to draft and execute on a strategic plan that aligns with the Code for America Digital Government Principles.

Official Brigade Chapter

  • An Official Brigade Chapter is a local group committed to organizing people, participating in the Code for America Brigade network, and working with municipal and community partners to execute on a strategic plan that aligns with the Code for America Digital Government Principles and Citizen Engagement Standards.
  • Official Brigade Chapters have a leadership teams, a consistent hack night, a strategic plan, municipal partners, and community partners.
  • Strategic plans allow Brigades to set yearly goals that align with the Code for America Focus Areas, and ensure their activities map to the improvement of their city offices.
  • Community partners are other local non-profit organizations that align with Brigades in their mission, members, and activities. Through strategic collaboration on events and projects, these partners can help expand the reach of Brigade activities and diversify their membership.

As of October 2014, there are 50 Official Brigade Chapters and 92 Brigade Network Members. We are adding new chapters and members every week.

For more on what it means to be a Code for America Brigade, check out the Brigade Website.
