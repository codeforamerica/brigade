1. Local Fundraising Guide

Donations vs. Sponsorships

I’ve heard that Code for America can help intake donations for my local Brigade. Are donations the same as sponsorships?


Short Answer:No, sponsorships and donations are not the same thing. Code for America can help Captains and Stage 2 leaders intake donations for their Brigade.

Long Answer: Donations are tax-deductible and cannot have business benefits for the donor (giving) company. Sponsorships are not tax-deductible, require a signed agreement, and include specific deliverables. Code for America can help Captains and Stage 2 leaders intake donations for their Brigade. Your local fundraising will be through donations, not sponsorships. Please clarify this with anyone who wants to provide resources for your Brigade.

2. Brigade Donations

What are the steps for securing a donation for my Brigade?
  1. Donor writes check // donor uses the online Code for America donation form and indicates your Brigade name under “What inspired your donation?”
    2. a. Checks should be made payable to “Code for America Labs.” Please include “for X City Brigade” in the memo.
    b. The mailing address for Code for America is:
    155 9th Street, San Francisco, CA 94103
  2. Code for America receives check
    3. a. The Code for America finance team will receive the check. - If under $1,000, full amount will be available for you to spend
    - If over $1,000, please first reach out to the Brigade staff (brigade-info@codefoamerica.org) Code for America to confirm how this will be spent and receive approval in advance.
  3. Funds will be added to your Brigade account to spend against. Note: these funds will be spent down before drawing against any of your Code for America HQ-approved budget.
  4. You can access funds by submitting expenses against the funds in Expensify using ths Support Reinbursement Process.
