Microsoft Bizspark / Azure

What is Bizspark / Azure?

BizSpark is a program that Microsoft offers to small organizations which includes many common Microsoft products and free Azure credits for hosting.

What do you get?

Through Bizspark, your Brigade will get 12 months of many Microsoft products including Microsoft Office, Microsoft Developer Network (including Visual Studio and SQL Server), and Microsoft Windows.

In addition, up to five of your developers will receive a $150 monthly credit for Microsoft Azure that they can apply to their project.
