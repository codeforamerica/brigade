Donated and Free Services

Building civic technology is tough, epecially with volunteers. Code for America has gotten a bunch of donated services for your Brigade to use. Have your Brigade leadership contact us at HQ to get access.

    Hosting

  • Microsoft Azure - $150 a month for three years in donated credits - Instructions
  • Amazon AWS - $5000 in donated credits for one to two years.
  • Heroku - Has a great free tier perfect for experiments.
  • Github Pages - Free and easy for static sites.

    Developer Resources

  • myBalsamiq - Donated access to this easy prototyping tool.
  • README.io - Great looking documentation for your projects - Forum post

    Mapping

  • CartoDB - Free upgraded accounts for this mapping service - Forum post
  • ArcGIS Online - Instructions

    Project Management

  • GitHub Issues - The most popular way to manage open source software projects. How To.
  • Waffle.io - A free project management tool based on GitHub Issues.
  • Hackshop by Waffle.io to help hackathon organizers and participants.
  • Trello

    Brigade Website Template

  • Classic Brigade Site - A Github Pages hosted simple site originally by Code for Hampton Roads.
  • Laddr - A really powerful Brigade management platform. Built by Code for Philly.

    Event Organizing

  • Meetup - Donated Meetup group for your Brigade. Nearly every Brigade uses this.
  • Eventbrite - Free event service.

    Open Data Portals

  • OpenDataSoft Open Data Portal - Forum post
  • Socrata Open Data Portals
  • CivicData.com

    Chat

  • Slack with Slackin or Slackinvite.me for autoinvites. There is a form at https://{yourteam}.slack.com/nonprofit to apply for a free upgrade. Contact us for a copy of our nonprofit certification letter.
  • Flowdock

    More

  • Awesome Civic - A crowdsourced list of civic tools.
  • Numbers - Counting the Brigades, People, and Projects.
