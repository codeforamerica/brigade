Code for America Brigade: Local Fundraising Toolkit

  1. This doc is for Brigade Organizers who are managing money for the Brigade, or hoping to find some.
  2. Both Official Brigade Chapters and Brigade Network Members should find this document helpful.
  3. If we left anything out, please email brielle@codeoforamerica.org

Donations & Sponsorships

  • Code for America can help Brigades intake donations for their Brigade. Donations are tax deductible and cannot have business benefits for the donor (giving) company.
  • Sponsorships are not tax-deductible, require a signed agreement, and include specific deliverables. Code for America can help Captains and Stage 2 leaders intake donations for their Brigade. Your local fundraising will be through donations, not sponsorships. Please clarify this with anyone who wants to provide resources for your Brigade.
    • Here is the process you will follow:
    1. Donor writes check // donor uses the online Code for America donation form and indicates your Brigade name under “What inspired your donation?”
      • Checks should be made payable to “Code for America Labs.” Please include “for X City Brigade” in the memo.
      • The mailing address for Code for America is:
        155 9th Street, San Francisco, CA 94103
    2. Code for America receives check
      • If under $1,000, full amount will be available for you to spend
      • If over $1,000, please first reach out to the Brigade staff (brigade-info@codefoamerica.org) Code for America to confirm how this will be spent and receive approval in advance.
    3. Funds will be added to your Brigade account to spend against. Note: these funds will be spent down before drawing against any of your Code for America HQ-approved budget.
    4. You can access funds by submitting expenses against the funds in Expensify using the Code for America Brigade Support Reimbursement Process.
  • Code for America can help you process Fiscal Sponsorships
    1. Raising funds to expand your Brigade activities? Planning to work on outside projects through the Brigade? Working on projects beyond a volunteer capacity (ie compensating people)?
      2. This requires three things
      • Confirmation that CfA is able to support this additional work. Email brigade-info@codeforamerica.org to start this conversation.
      • Based on the nature of your Sponsorship, Code for America will help you coordinate a Formal Fiscal Sponsorship agreement
      • Based on the nature of your Sponsorship, Code for America will help you set up a bank account for your local Brigade
    2. Code for America will then accept the Sponsorship funds on your behalf and write a check directly to your local Brigade's bank account, which you can use to pay for expenses directly.
